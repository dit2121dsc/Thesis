Due to the recent increase in the number of devices connected to different networks, information traffic has increased significantly. As a result of this, the number of threats has also increased. Thus, other works proposed intrusion detectionCSO systems (IDSCSO) to protect sensitive user data. IDSTerms are responsible for identifying malicious data flows and reporting possible attacks. However, the first IDSCSO have based on detecting attacks on signatures. Therefore, IDSCSO cannot keep up with the constant evolution of existing attacks. Hence, techniques such as Machine LearningCSO (ML) have become allies of this system type to ensure its effectiveness. The use of ML represents a significant advance in the development of IDSCSO, but there are still open questions about the ability to detect attacks on different isolated networks. Therefore, the present work proposes a Federated Learning (FL) scheme with sampling and attribute selection methods for Distributed Denial-Of-Service (DDoSTechnology) classification. Furthermore, we propose to combine the FL scheme with the Energy-based Flow Classifier (EFCTerms) algorithm building an ensemble model capable of identifying malicious agents. We evaluated whether using an ensemble can extract different types of information during the ML process. This work represents ongoing research with results under development.
