The paper focuses on a new type of interactive learning content for SQLCSO programming -worked examples of SQLCSO code. While worked examples are popular in learning programming, their application for learning SQLCSO is limited. Using a novel tool for presenting interactive worked examples, Database Query Analyzer (DBQATerms), we performed a large-scale randomized controlled study assessing the value of worked examples as a new type of practice content in a database course. We report the results of the classroom study examining the usage and the impact of DBQATerms. Among other aspects, we explored the effect of textual step explanations provided by DBQATerms.
